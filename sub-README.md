One of my biggest contributions that I added to TERRA was helping with the general structure and logic behind our database. I contributed to the SQL data and helped think about how our tables including country, country_year_data, climate_event, risk_assessment, etc would connect to the general structure and purpose of the app we are building. ### REST API Contribution

Another major contribution I made was working on most of the app’s REST API routes but not all of them. These routes are all important because they are what help connect the Streamlit front end to the backend and MySQL database. I was able to work more on the logic behind how the data should be created, read, updated, or deleted through routes which overall helped make TERRA function as a full on app.
